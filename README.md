  G++ setup (The very first thing you should do)
  
  C++.sublime-build (ONLY FOR VIEW YOUR TEXT)
  
  G++.sublime-build (TO ENTER INTO THE CONSOLE)
  
  Test.cpp (Just test program)
  
  encoding of C++ characters in all languages (-)

In folder Python :
